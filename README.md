### **Система управління книжковим магазином (Exam Project)**

#### **Технологічний стек:**
Python 3.13

PostgreSQL

SQLAlchemy

python-dotenv

#### **Структура проєкту:**
Проєкт розділений на модулі для легшої підтримки:

**main.py** - точка входу

**cli.py** - інтерфейс користувача; обробка введення та вивід таблиць, меню користувача

**repositories.py** — усі SQL-запити (CRUD, звіти)

**models.py** — автоматичне зчитування структури БД (Reflection) через autoload_with

**db.py** — безпечне підключення до БД через змінні оточення

**validators.py** — функції для перевірки email, дат та числових значень

**services.py** — сервіс експорту звітів у формат CSV

requirements.txt — мінімальний набір бібліотек (SQLAlchemy, Psycopg 3, Dotenv)

#### **Інструкція із запуску**
1. Підготовка бази даних

Створіть базу в PostgreSQL.

Виконайте скрипт Photobook_Exam.sql для створення таблиць та наповнення даними (50 книг, 10+ співробітників, 230+ продажів).

2. Запуск

Виконайте: python main.py

#### **Функціональні можливості**
1. А**дміністрування**

Книги та Співробітники: додавання та редагування з миттєвим відображенням змін.

Soft Delete: при видаленні книга або співробітник позначаються як "архівні" (is_deleted), що зберігає цілісність історичних звітів.

2. **Операційна діяльність**

Реєстрація продажу: вибір книги та продавця за ID/ISBN з перевіркою існування.

Коригування: можливість змінити фактичну ціну продажу або повністю видалити помилковий запис про продаж із бази.

3. **Аналітика та звітність (11 звітів)**

Програма формує звіти безпосередньо через SQL-агрегації:

- Фільтри: за датою, за періодом, по конкретному співробітнику.

- ТОП-показники: найкраща книга, найкращий продавець, популярний жанри та автори.

- Фінанси: розрахунок виручки, собівартості та чистого прибутку.

- Експорт: будь-який сформований звіт можна зберегти в папку export/ у форматі CSV.

#### **Інтерфейс користувача**
Реалізовано зручне ієрархічне меню:

**Адміністратор** (Робота з довідниками).

**Операційна діяльність** (Продажі та каталог).

**Аналітика** (Фінансові та статистичні звіти).
